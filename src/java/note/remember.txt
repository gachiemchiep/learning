オブジェクト指向を極めよう
======================

「修飾子」　class サブクラス名 extends スーパークラス名 {
	メンバ変数の定義
	メソッドの定義
}

メソッドの再定義
	オーバーライド

this	: 現在のオブジェクト
super	: 現在のオブジェクトのスーパークラスのオブジェクト

サブクラスの作成できないクラス
	
	final class クラス名 {

	}

インタフェースの定義（メソッドの使い方を統一にする）

	「修飾子」interface インタフェース名 {
		定数の定義
		抽象メソッドの定義
	}

インタフェースの実装

	class クラス名 implements インタフェース名 {
		メソッドのオーバーライド
	}
