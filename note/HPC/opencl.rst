Set up
--------

OpenCL library is already included inside cuda took-kit.

.. code-block:: html

    $ clinfo
    clinfo: /usr/local/cuda-8.0/lib64/libOpenCL.so.1: no version information available (required by clinfo)
    ....
    Device Version                                  OpenCL 1.2 CUDA
